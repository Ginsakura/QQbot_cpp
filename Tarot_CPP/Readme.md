## Tarot

使用方式：`./Tarot.exe [QQ号]`

传入操作人，自动计算抽取塔罗牌，每次抽取三张。

数据存储至SQLite数据库：`./Tarot_Data/Tarot_Data.db`

使用第三方库[`sqlite3`](https://github.com/Ginsakura/QQbot_cpp/tree/main/Library)
